================================================================================
ThrowTrajectoryCorrector（投擲軌道補正）まとめ
================================================================================
対象スクリプト: Assets/Resources/Scripts/ThrowTrajectoryCorrector.cs
主な使用プレハブ: Assets/Resources/prefabs/NO BOM.prefab

--------------------------------------------------------------------------------
1. 目的
--------------------------------------------------------------------------------
キャラクターが投げたオブジェクトについて、
「投擲方向がターゲット方向に近いときだけ」軌道を補正し、命中しやすくする。

・移動速度は一定（重力なし）
・軌道は Y 軸固定（XZ 平面のみ移動。補正あり・なし共通）
・軌道は投擲時に一度だけ決定（飛行中に再計算しない）
・ターゲットが後から動いても、ゴール地点は更新しない
・Dynamic Rigidbody で移動し、壁に当たると誘導終了

--------------------------------------------------------------------------------
2. 基本仕様
--------------------------------------------------------------------------------
【投擲方向】
  P0 … 手から離れた位置
  P1 … 次の FixedUpdate の位置
  投擲方向 = normalize(P1 - P0) の XZ 成分

【補正判定】
  投擲方向とターゲット方向の角度が 30 度未満 → ベジェ曲線で補正
  30 度以上 → 直進

【ゴール位置】
  X, Z … ターゲット Bounds 中心
  Y    … Goal Height（既定 1）

--------------------------------------------------------------------------------
3. ターゲット設定
--------------------------------------------------------------------------------
  Find Target If Unset … ON
  Find By Tag … ON
  Target Tag … Boss

  プレハブからシーンの BOSS を直接 Throw Target に入れられないため、
  投擲時にタグ検索で解決する。

--------------------------------------------------------------------------------
4. Inspector 主要項目
--------------------------------------------------------------------------------
【速度】
  Use Throw Measured Speed … 投擲実測速度を使うか
  Flight Speed … 固定速度
  Speed Multiplier … 倍率

【物理】
  Stop Flight On Collision … 壁接触で誘導終了
  飛行中: Dynamic RB + Continuous Dynamic + Y/回転固定
